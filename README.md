# Codigo 

## Importación y Configuración

1. ChatterBot: Es la clase principal que crea la instancia del bot.
2. ListTrainer: Es el método de aprendizaje que permite al bot aprender a partir de estructuras de datos tipo lista (preguntas y respuestas directas).

## Proceso de Entrenamiento Personalizado

En esta fase, convertimos al bot genérico en un experto en Sistemas Digitales. Se cargan las definiciones de compuertas lógicas, sistemas binarios y componentes como el temporizador 555. Cada par de frases en la lista fortalece la base de conocimientos.

## Bucle de Interacción (Feedback Loop)

El programa entra en un ciclo infinito que permite la conversación en tiempo real. La función get_response no solo busca una respuesta, sino que analiza la intención del usuario comparándola con la base de datos entrenada.
